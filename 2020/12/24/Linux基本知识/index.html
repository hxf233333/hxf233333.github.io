Linux基本知识

Linux常用命令分类

VI的使用

  • vi a.txt

    • 打开或者新建打开a.txt文件

  • 命令模式

    • 在命令模式下可以通过a/A/i/I/o/O中的任意一个键进入插入模式

    • 在命令行模式下通过输入“:”进入末行模式

      • 按Esc返回命令模式

      • :q!

        • 不保存退出

      • :wq!

        • 保存退出

      • :w filename

        • 另存为

      • :m,n w filenname

        • 将m行到n行之间的内容另存为

      • :set nu

        • 显示行号

      • :set nonu

        • 取消行号

    • y

      • 选择复制

    • yy

      • 复制当前行

    • nyy

      • 从当前行开始复制多行

    • p

      • 粘贴至光标后

    • d

      • 选择剪切

    • dd

      • 剪切当前行

    • ndd

      • 从当前行开始剪切多行

    • nG

      • 定位到n行

    • u

      • 撤销操作

  • 插入模式

    • 在插入模式下可进行文本输入，按Esc切换到命令模式哦

磁盘管理

  • df

    • 参数

      • -h

        • 默认的格式输出文件

      • -m

        • 以M为单位输出文件系统的占用情况

      • -k

        • 以k为单位输出文件系统的占用情况

      • -T

        • 显示磁盘分区的文件系统类型

    • 案例

      • 查看当前磁盘分区并人性化输出

        • df -hT

      • 输出文件系统的inode信息

        • df -ihT

  • du

    • 参数

      • -s

        • 按照单位KB的方式显示文件或整个目录的大小

      • -b

        • 按照字节的单位显示文件大小

      • -sh

        • 按照更加人性化的方式显示文件大小

      • -sm

        • 按照MB的方式显示文件或目录的大小

    • 案例

      • 显示当前目录下所有文件大小

        • du -sh./*

      • 显示某个目录的大小

        • du -sh /etc/

  • sync

    • 内存数据同步写入磁盘

  • mount

    • 参数

      • -a

        • 加载文件/etc/fstab中设置的所有设备

      • -n

        • 不将加载信息记录在/etc/mtab文件中

      • -r

        • 按照只读的方式加载设备

      • -w

        • 按照可读写的方式加载设备

    • 案例

      • 将光盘镜像文件挂载在Linux系统中

        • mount -o loop -tiso9660.iso /ixdba

      • 在一台Linux中按照只读的方式挂载另一台Linux系统上共享的分区

        • mount -t nfs -o ro IP:/home/ixdba/mnt/nfs

性能监控

  • uptime

    • 显示系统运行时间信息

  • dmesg

    • 显示系统开机信息

  • uname

    • 打印系统软硬属性

  • free

    • 显示系统内存使用情况

  • ps

    • ps -aux

      • 显示当前用户进程信息

  • top

    • 实时显示系统中各个进程的资源占用情况

  • kill

    • kill -9 pid

      • 强行终止指定进程

网络设置

  • ifconfig

    • 参数

      • -a

        • 显示所有的网络接口信息

      • -s

        • 仅仅显示每个接口的摘要数据

      • -v

        • 方便查看错误信息

    • 案例

      • 在网卡enp0s3配制两个ip

        • ifconfig enpS33 ip netmask 255.255.255.0

      • 修改网卡的MAC地址

        • ifconfig enpS33 ether

  • scp

    • 参数

      • -r

        • 递归传输

    • 案例

      • 将A机器的A.TAR文件复制到B机器的/tmp

        • scp /home/a.tar.gz root@ip: /tmp

      • 通过-r的方式复制整个目录

        • scp -r local_folder remote_username@remote_ip:remote_folder
        • scp -r local_folder remote_ip:remote_folder

  • netstat

    • 案例

      • 显示路由信息

        • netstat -rn

      • 显示当前系统所有TCP连接

        • netstat -an

      • 显示目前已经连接的网络信息

        • netstat -tlnu

      • 显示当前系统处于连接状态的资源信息

        • netstat -atunp

  • traceroute

    • 参数

      • -i

        • 指定网口发送数据

      • -n

        • 直接使用IP地址而不是主机名

      • -v

        • 详细显示命令行执行过程

      • -x

        • 开启或者关闭对数据包的IP地址

      • -s

        • 设置本机主机发送数据包的IP地址

    • 案例

      • 跟踪从本机到访问百度的数据包发送过程

        • traceroute -l eth0 -s ip -w www.baidu.com

  • telnet

    • 案例

      • 登录到开启了telnet的远程主机上

        • telnet ip 23

      • 检测22端口是否打开

        • telnet ip 22

  • wget

    • 从网络下载一个文件并保存在当前目录

      • wget http://cn.wordpress.org/wordpress-4.9.4-zh_CN.tar.gz

  • ssh

    • ssh user@ip

      • ssh登录远程主机(默认连接用户的家目录)

文件管理和编辑

  • mkdir

    • 参数

      • -m

        • 对创建的目录设置访问权限

      • -p

        • 递归创建目录

    • 案例

      • mkdir -m 777 -p /home/user/user2

  • more

    • 参数

      • -d

        • 在屏幕底部显示友好信息比如more命令帮助

      • -p

        • 清除显示屏之前的内容

      • -c

        • 显示文件的时候，每次都清除屏幕之前的信息

    • 案例

      • more -c -10 x.txt

        • 每10行显示一屏

  • diff

    • 参数

      • -c

        • 显示全部内容并标记不同之处

      • -b

        • 忽略空格

      • -r

        • 如果是目录则会比较子目录的文件

      • -s

        • 文件相同的时候是文件的相同的部分

    • 案例

      • diff a.txt b.txt

        • 比较两个文件差异

  • cat

    • 参数

      • -b

        • 将文件中的所有非空行按照顺序编号，编号从1开始

    • 案例

      • cat a.txt > b.txt

        • 终端显示a.txt中内容并重定向到b.txt

  • grep

    • 参数

      • -c

        • 显示符合条件的行数

      • -i

        • 搜索的时候忽略大小写

      • -n

        • 在搜索结果中显示行数

    • 案例

      • grep hxf a.txt

        • 列出a.txt中包含“hxf”字符串的行

      • grep -ni hxf a.txt

        • 列出a.txt包含“hxf”的行，忽略大小写且标识行号

  • rm

    • 参数

      • -f

        • 忽略不存在的，系统不给出提示

      • -i

        • 进行交互式删除（系统提示）

      • -r，-R

        • 将参数中列出的全部目录和子目录递归删除

    • 案例

      • rm -rf text

        • 递归不提示删除文件夹text内容

      • rm -i *.txt

        • 逐一提醒是否删除txt文件

  • touch

    • touch file

      • 新建一个不存在的文件

  • ln

    • 参数

      • -b

        • 删除，覆盖以前的链接

      • -d

        • 允许超级用户制作目录的硬链接（创建副本）

      • -f

        • 强制执行

      • -i

        • 交互模式，文件存在则提示用户是否覆盖

      • -s

        • 软链接（创建快捷方式）

    • 案例

      • ln -s a.txt a

        • 为a.txt创建软链接a，如果a.txt丢失，a将失效

      • ln a.txt text/

        • 将文件链接为另一个目录中的相同名字，无论是软链接还是硬链接，文件都保持同步变化

      • ln -sv /text1/ /text2/

        • 给目录创建软链接(目录只能创建软链接且为绝对路径)

  • cp

    • 参数

      • -p

        • 保留文件的修改时间和存取权限

      • -i

        • 复制的时候提示是否覆盖

      • -a

        • 复制目录的时候保留所有信息

      • -r

        • 递归复制目录

    • 案例

      • cp -r ./* /home/hxf

        • 将当前目录所有内容复制到/home/hxf下

  • find

    • 格式

      • find pathname -option name [-print -exec -ok]

        • pathname：表示所要查找的目录路径,例如”.”表示当前目录，”/”表示根目录
        • -print:将find找到的文件输出到标准输出
        • -exec/-ok:对找到的文件执行exec这个参数所指定的shell命令，相应的形式为：-exec command {} ; 将查到的文件进行command操作，”{}”就代替查到的文件

    • 参数

      • -name

        • 根据文件名进行查找,区分大小写精确查找

      • -iname

        • 根据文件名查找，不区分大小写

      • -user

        • 根据文件所有者查找文件

      • -group

        • 根据所有者所在的组查找

      • -type

        • 根据文件类型查找文件

          • 普通文件：f
          • 目录文件：d
          • 链接文件：l
          • 块设备文件：b
          • 字符设备文件：c
          • 管道设备文件：p
          • 套接字文件：s

      • -size

        • 根据文件大小查找文件

          • +n

            • 表示大于n字节的文件

          • -n

            • 表示小于n字节的文件

          • n

            • 表示等于n字节的文件

      • -a，-o，-not

        • 对不同的条件进行与或非

    • 案例

      • find . -name file* -a -size +1000 -exec cat {} \

        • 查找当前目录下前缀名为file且大于1KB文件，并在终端打印

  • mv

    • 格式

      • mv [参数] 源文件或者目录 目标文件或者目录

    • 参数

      • -b

        • 覆盖同名文件

      • -f

        • 强制执行

      • -i

        • 交互模式，文件存在则提示用户是否覆盖

      • -t

        • 移动多个源文件时，这时目标目录在前，源文件在后

    • 案例

      • mv test1.txt hxf.txt

        • 重命名test1.txt

      • mv test1.txt ../text/

        • 移动test1.txt文件

压缩与解压

  • tar

    • 案例

      • tar -cvf hxf.tar test1.txt test2.txt

        • 仅打包，不压缩

      • tar -zcvf hxf.tar.gz test1.txt test2.txt

        • 打包后，以 gzip 压缩

      • tar -zcvf hxf.tar.bz2 test1.txt test2.txt

        • 打包后，以 bzip2 压缩

      • tar -ztvf hxf.tar.gz

        • 查看包里有哪些文件

      • tar -zxvf hxf.tar.gz

        • 解压缩

  • zip

    • 案例

      • zip hxf.zip test1.txt test2.txt

        • 压缩zip格式

      • unzip hxf.zip

        • 解压缩

  • rar

    • 案例

      • rar a hxf.rar test1.txt test2.txt

        • 压缩rar格式

      • rar x hxf.rar

        • 解压缩

  • dd

    • 案例

      • dd if=/home/user1 of=/home/user2

        • 将/home/user1备份到/home/user2

  • man
